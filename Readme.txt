Программа управления МК с помощью FreeRTOS

Назначения выводов
PA2(input) - Btn  (кнопка)

PA3(output)  - Led1 (моргающий светодиод)
PA4(output)  - Led2 (моргающий светодиод)
PA6(output)  - Led3 (cветодиод включаемый кнопкой, через "семафор")
PB10(output) - Led4 (светодиод, регулируемый с помощью ШИМ, TIM2_Channel3)

PB0  - ADC1_IN8 (меряет напряжение с потенциометра)

PA10 - USART1_RX
PA9  - USART1_TX

Baud Rate 115200 Bits/s
Word Lenght 8 Bit
Parity None
